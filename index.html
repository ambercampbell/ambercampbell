  • Menlo Innovations

    Software Engineering Intern | Ann Arbor, MI | | June 2023 - September 2023

    At Menlo, I worked on a web application designed to help healthcare professionals review prescriptions for Direct Oral Anticoagulants (DOACs). It identifies patients on DOACs and applies logic rules to assess prescription appropriateness. While similar tools exist in Epic EHR, Clover was developed as a stand-alone app using FHIR interoperability standards, allowing seamless integration with Epic and other EHR systems for broader application.

    The application addresses prescription errors, which occur in 10-15% of DOAC prescriptions, by providing nurses and pharmacists with an efficient tool to review and correct errors. The dashboard displays alerts associated with specific patient prescriptions, offering detailed patient information and allowing users to take action, such as snoozing or resolving alerts.

    The back-end utilizes a daily cron job to generate alerts by pulling and parsing patient data from EHR systems. To comply with HIPAA regulations, it stores only FHIR IDs, fetching patient data as needed. The app is built using HTML, CSS, JavaScript, Python, Django, PostgreSQL, and AWS, and is optimized for deployment within Epic Hyperspace without the need for complex front-end frameworks.

Memory Manager

Course Project: Advanced Operating Systems

I co-designed and implemented a pager for managing virtual address spaces within an operating system kernel. My work involved overseeing key memory management functions, including the allocation of address space, management of memory and swap files, and the development of system calls to create, copy, and destroy address spaces, enabling seamless interaction between user applications and the pager. I implemented an interrupt handler to efficiently manage page faults, ensuring that pages were correctly loaded from swap or file storage into physical memory. By supporting both swap-backed and file-backed virtual pages, I optimized memory usage and ensured robust process isolation.

Scientfic Theories as Bayesian Networks

Utilizing Python and the pyAgrum library, I created a custom class for converting between different Bayesian network representations and developed algorithms to mutate networks towards an ideal structure. This work deepened the understanding of scientific inference and theory evolution.
